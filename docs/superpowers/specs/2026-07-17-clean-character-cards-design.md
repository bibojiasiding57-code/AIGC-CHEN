# 角色介绍无色块卡片设计

## 目标

保留当前角色介绍区的四卡横排结构，撤销单主角海报方案。删除人物卡片中的红、蓝、绿等色块及染色效果，让人物以原始图片颜色呈现，同时保留角色名称和英文说明。

## 视觉方案

- 保留白色圆角展柜、顶部品牌与搜索区、四卡横排、底部标签和前后切换按钮。
- 每张卡片改为白色或极浅暖白底，不使用彩色背景、渐变、彩色遮罩或混合染色。
- 人物原图完整显示，沿用现有裁切位置，并恢复素材原始色彩。
- 角色名称和英文说明放在图片下方独立的白色信息栏中，使用深色文字，避免覆盖人物。
- 卡片之间继续使用一致的圆角、间距和轻微边界，让四张图片保持清晰分组。

## 交互与数据

- 角色数据、搜索过滤逻辑和顺序保持不变。
- “上一组 / 下一组”继续横向滚动角色列表。
- 搜索无结果时继续显示现有空状态。
- 不增加新页面、弹窗或额外角色交互。

## 响应式表现

- PC 端维持一屏约四张卡片。
- 中等屏幕维持约三张卡片。
- 手机端保留单卡为主的横向滑动浏览，名称与英文说明始终可见。

## 实现范围

- 仅调整角色介绍区的卡片结构样式和必要的标记。
- 删除卡片对角色颜色变量的依赖，以及彩色背景、渐变与 `mix-blend-mode` 效果。
- 不修改首屏、作品案例、互动体验和联系方式。

## 验收标准

1. 四张可见角色卡保持横排。
2. 所有人物恢复原始图片颜色。
3. 卡片内没有红、蓝、绿等色块或染色遮罩。
4. 角色名称和英文说明清晰保留，并位于独立白色信息栏。
5. 搜索、前后切换和手机横滑继续正常工作。
6. 页面无新增控制台错误，桌面和手机布局没有溢出或文字裁切。
